Dear {{name}} 開源貢獻者：

OSCVPass 開源貢獻者快速通關專案從 2019 年底開始運作，目前已經運作邁入第五年，總共有 596 人次申請，這四年也跟七個研討會合作 17 場次。

OSCVPass 也將有意願公開並通過申請的開源貢獻者案例呈現在網站，讓有興趣的朋友們能夠觀看查閱。目前的分類架構依照大家貢獻專案的內容，暫時區分為 11 類：分別為開源、社群貢獻、程式語言與設計、資安 / 區塊鏈、語言、翻譯、影像辨識 / 聲音辨識、APP、Chatbot / API、Linux Kernel 與發 PR 修 Bug 等。

累計至 2022 年底仍然以「發 PR 修 Bug」為最大宗，共計有 32 % 使用該類別來作申請；其次是「開源」，共計有 22 % 使用該類別來作申請；第三高的則是「社群貢獻」，共有 13% 使用該類別來作申請，然而其餘八類中以翻譯提升的比例最多，也表示原本除了 Code 上的貢獻外也多了其他非 Code 類的開源貢獻。

除了 OSCVPass 社群串連 之外，OCF 做為台灣開放 / 開源的推手，一直走在議題的最前線，希望不分界線地與來自不同領域的人們牽手，一同為台灣的數位韌性與數位民主貢獻心力。

九年來，OCF 積極突破同溫層：我們輔助台灣企業首次取得國際開源認證、啟動「企業數位人權排名」；我們協助立法院開放國會行動方案，提升公私協作能力；我們與法律、藝術、科技社群合作，舉辦工作坊及跨界藝術派對，讓民眾走入無邊界的開放 / 開源領域，與企業、民間與政府共同打造科技能為人所用的數位世界。

在推動社群知識共享互助之際，我們更將開源貢獻者們送上國際舞台，透過民間交流讓世界看見這座島嶼的勃勃生機。OCF 與各國 NGO 組織串連、積極參與海外數位人權及開源大會，分享台灣的數位民主案例，更將世界各處的議題帶回台灣。
