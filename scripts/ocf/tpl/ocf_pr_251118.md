# 讓音樂成為可 fork 的 repo：一起在現場驗證共享與再製

去年，我們和你一起見證林強與 Luca Bonaccorsi 以公眾授權釋出《[時間浸漬](https://www.youtube.com/playlist?list=PLFuYOsppHDrlAZhOyeqC7cOaKT3MKkbBo)》，讓音樂像原始碼一樣可[自由下載](https://10years.ocf.tw/bioerosion-album.html)、再製、散佈，甚至被用於 Taiwan Plus 紀錄片、兩廳院宣傳與多部創作者短影音，證明開源方法能在文化場景被 fork 與 merge。

開放文化基金會 OCF [十週年](https://10years.ocf.tw/)，我們把開源精神帶進更多文化現場，從北流 JAM JAM ASIA 的「開放音像派對」到最新延伸專案《明日音》，邀你把對開源的想像帶進一場以共享為核心的現場實驗。

開源不只寫程式，還能長出聲音的新分支 ...

今年，這條聲音脈絡繼續延伸：Mong Tong 與新媒體團隊 XTRUX 重新取樣《時間浸漬》的開放素材，以遊戲引擎把聲音與即時影像串起來，讓舞台成為可互動的「實體 repo」。

2025/11/30（日），臺北流行音樂中心 SUB LIVE，我們不開會、不簡報、不放投影片，只想跟你邊聽邊聊、交換近況；現場將設有主題攤位與餐車，是社群久違的線下團聚。晚間由迷幻電子雙人組 Mong Tong 帶來現場演出！

《明日音》限量彩膠與門票正於 [FlyingV 募資中](https://www.flyingv.cc/projects/35562)，此次亦為 SUB LIVE 唯一專場，錯過不再！
